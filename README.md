A simple job board built with Next.js 15, where users can browse and filter jobs, view job details, and—after logging in—create, edit, and delete their own listings. The project uses PostgreSQL with Prisma, NextAuth for authentication, Tailwind CSS for styling, and Zod for validation. It showcases modern full-stack development with a clean and responsive design.


## نصب و راه‌اندازی

1. نصب وابستگی‌ها:

```bash
npm install
```

2. اجرای برنامه در حالت توسعه:

```bash
npm run dev
```
